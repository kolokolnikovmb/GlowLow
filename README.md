# GlowLow

SwiftUI MVP for daily mood check-ins with local-only storage.

## Stack

- iOS 17+
- SwiftUI
- SwiftData
- UserNotifications

## Generate project

1. Install Xcode and [XcodeGen](https://github.com/yonaskolb/XcodeGen).
2. Run `xcodegen generate`.
3. Open `GlowLow.xcodeproj`.

## Notes

- Data is stored locally with SwiftData.
- Notifications are local-only.
- The repo includes source files and `project.yml`; `.xcodeproj` is intentionally not committed.
